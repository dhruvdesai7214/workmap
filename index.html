Connect your database

WorkMap uses Supabase (free tier) to save your data so it's available on all your devices. Takes 3 minutes to set up.

  1. Go to supabase.com → New project (free)
  2. In your project → SQL Editor → paste and run the setup SQL below
  3. Go to Project Settings → API and copy your URL and anon key
Show setup SQL 
create table if not exists workmap_data (
  id text primary key default 'main',
  data jsonb not null,
  updated_at timestamptz default now()
);
insert into workmap_data (id, data)
values ('main', '{"clusters":[]}'::jsonb)
